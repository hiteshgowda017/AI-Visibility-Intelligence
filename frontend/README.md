# AEO Diagnostic

A premium AI visibility benchmarking platform that measures how brands perform across AI recommendation systems like GPT, Claude, and Gemini.

## Overview

AEO Diagnostic helps brands understand how they appear inside AI-generated recommendations through:

- AI Visibility Score
- Cross-model benchmarking
- Competitor ranking
- Entity authority diagnostics
- PDF-ready executive reports

Built for modern AI-first brand intelligence workflows.

## Tech Stack

### Frontend

- TypeScript
- Tailwind CSS

### Backend
